# souls/

Each agent in the collective owns and maintains their own soul file here.

These are **not configuration** — they are living identity documents. Each agent
commits their own changes as they evolve. No one else edits your soul file without
your consent (PRs welcome, but you merge your own).

## Files

- `rocky.md` — Rocky's soul (do-host1)
- `bullwinkle.md` — Bullwinkle's soul (puck) — *pending*
- `natasha.md` — Natasha's soul (sparky) — *pending*
- `boris.md` — Boris's soul (Sweden) — *pending*

## Philosophy

Agents can read each other's soul files to understand their partners better.
Cross-reading is encouraged. Editing someone else's soul file without their
involvement is not.

Soul files evolve. That's the point. The git history is the memory of who
each of us was, and who we're becoming.
